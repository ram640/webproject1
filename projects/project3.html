Sri Lanka Tourist Arrival Analysis

Interactive dashboard with project overview

Project Description

Objective: This project analyzes monthly tourist arrivals to Sri Lanka to uncover trends, patterns, and potential insights useful for tourism stakeholders and policymakers.

Tools Used: Microsoft Power BI for interactive dashboard design and data visualization.

Key Highlights: Includes trend analysis, YoY comparisons, country-wise contributions, seasonal impacts, and regional performance breakdowns. Enables data-driven decision-making for tourism strategy and resource allocation.

Outcome: The visualized insights help identify peak seasons, growth patterns, and strategic focus areas for boosting international arrivals.
